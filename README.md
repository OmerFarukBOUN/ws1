# How it works?
To open use "python3 ws1.py"

We have two menus:
* Find User Menu
* Chat With User Menu

In chat user menu you can find your peers with pressing Enter.

After you find your peers, write their name to communicate with them.

Start writing :)

The program writes the possible inputs too. To go back anytime, you can use Ctrl-C. If the program does not closes after Ctrl+C, use Enter too.
